# Task Manager Application

A secure, full-stack task management system designed to keep your productivity high and your data safe.

## 🚀 Features
* **Secure Authentication:** User accounts protected with **JWT** (JSON Web Tokens) and passwords hashed via **Bcrypt**.
* **Automated Notifications:** 
    * Instant welcome email upon registration.
    * Automated email reminders for every task you add.
* **Task Management:** Create, track, and manage your future goals effortlessly.
* **Reliable Architecture:** Built with Node.js, Express, and MySQL for robust data handling.

## 🛠 Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Security:** JWT, Bcrypt
* **Email Service:** [Insert Email Service Name, e.g., Nodemailer/SendGrid]

## 💻 How to Run Locally
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your `.env` file with your database and email credentials.
4. Run the server: `npm start`
